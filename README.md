# Detalk
사용자 참여를 통해 제품 추천 및 피드백 공유가 가능한 커뮤니티 기반 서비스로 주로 영어권 사용자들을 타겟팅한 웹 서비스입니다.

- [서비스 바로가기](https://detalk.net)

## 시작하기
